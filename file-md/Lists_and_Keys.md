# Lists & Keys ב־React

## רינדור מערך

``` javascript
items.map(item => (
  <div key={item.id}>{item.name}</div>
))
```

------------------------------------------------------------------------

## למה צריך key?

כדי ש־React ידע מה השתנה ברשימה.

❌ שימוש גרוע:

``` javascript
key={index}
```

✅ עדיף מזהה ייחודי.

------------------------------------------------------------------------

## כלל חשוב

key חייב להיות ייחודי בין האחים.
