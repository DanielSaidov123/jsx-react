# סיכום useEffect ב־React

## מה זה useEffect?

Hook שמאפשר להריץ קוד אחרי רינדור הקומפוננטה. משמש ל־API, טיימרים,
מאזינים ועוד.

תחביר בסיסי:

``` javascript
useEffect(() => {
  // קוד
}, [dependencies]);
```

------------------------------------------------------------------------

## הרצה פעם אחת (componentDidMount)

``` javascript
useEffect(() => {
  console.log("Runs once");
}, []);
```

------------------------------------------------------------------------

## הרצה כשמשתנה משתנה

``` javascript
useEffect(() => {
  console.log("count changed");
}, [count]);
```

------------------------------------------------------------------------

## ניקוי אפקט (cleanup)

``` javascript
useEffect(() => {
  const timer = setInterval(() => {}, 1000);

  return () => clearInterval(timer);
}, []);
```

------------------------------------------------------------------------

## נקודות חשובות

-   בלי מערך תלויות → רץ בכל רינדור
-   \[\] → פעם אחת
-   תמיד לנקות מאזינים / טיימרים
