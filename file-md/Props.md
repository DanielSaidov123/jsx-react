# סיכום Props ב־React

## מה זה Props?

דרך להעביר מידע מקומפוננטה אב לקומפוננטה בן.

``` javascript
function Child({ name }) {
  return <h1>{name}</h1>;
}

<Child name="Dan" />
```

------------------------------------------------------------------------

## Props הם Read‑Only

אסור לשנות אותם בתוך הקומפוננטה.

------------------------------------------------------------------------

## העברת פונקציות דרך Props

``` javascript
function Child({ onClick }) {
  return <button onClick={onClick}>Click</button>;
}
```

------------------------------------------------------------------------

## שימוש נפוץ

-   העברת נתונים
-   שליחת אירועים למעלה
