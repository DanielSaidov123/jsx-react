# useMemo / useCallback -- סיכום קצר

## useMemo

שומר תוצאה של חישוב יקר.

``` javascript
const value = useMemo(() => compute(a, b), [a, b]);
```

------------------------------------------------------------------------

## useCallback

שומר פונקציה כדי למנוע יצירה מחדש.

``` javascript
const handler = useCallback(() => {}, []);
```

------------------------------------------------------------------------

רק כשיש בעיית ביצועים אמיתית. שימוש יתר → מיותר.
