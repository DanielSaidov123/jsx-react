# Conditional Rendering ב־React

## הצגה לפי תנאי

``` javascript
{isLoggedIn ? <Dashboard /> : <Login />}
```

------------------------------------------------------------------------

## שימוש ב־&&

``` javascript
{isAdmin && <AdminPanel />}
```

------------------------------------------------------------------------

## פונקציה רגילה

``` javascript
if (loading) return <Spinner />;
```

------------------------------------------------------------------------
